About this game

We worked together as a class to build this project. It uses Twine, which lets you draw a set of nodes to make a "choose your own adventure" story. In this project we used variables, loops, and conditions. We also learned how to format text and add custom fonts.

Scratch Game - a MAZE ing

A Mazing Game

By Miriam

About this project

To do this project, I used Scratch, which is both easier and harder than people might think to use. It's easier because you don't have to memorize any commands, but you still have to have a mindset that will guide you towards the goal.

The hardest part was tuning up the difficulty level so it was possible to beat without being so hard it was no fun.

Twine Blackjack

blackjack.html

By Jonathan

About this project

I used a technology called Twine, which is an online programming tool. It is designed to make adventure games. I modified it to make a form of blackjack. It was harder than I anticipated. I had to cut a lot of corners to make it work.

I learned some programming concepts like if / else statements, variables, and formatting.

Blender Model - Still life with furniture

Furniture image

By Daniel

About this project

I built this image with Blender, a 3D modeling program. I learned basic modeling functions such as rotate, extrude, and translate. I learned how to set up custom materials to give objects color, and how to set up lights and a camera.

3D modeling is a lot more complicated than I thought, but it is a lot of fun.
